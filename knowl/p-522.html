Also recall (from Unit 1) that for a function, \(f\text{,}\) to have an inverse, \(f^{-1}\text{,}\) it must be one-to-one, meaning no horizontal line intersects the graph more than once. Since this is not true for trigonometric functions, they do not have inverses. We need functions to be one-to-one because both \(\sin\frac{\pi}{4}=\frac{\sqrt{2}}{2}\) and \(\sin\frac{3\pi}{4}=\frac{\sqrt{2}}{2}\) but if we took the inverse of sine, would we wouldn't know to use \(\sin^{-1}\left(\frac{\sqrt{2}}{2}\right)=\frac{\pi}{4}\) or \(\sin^{-1}\left(\frac{\sqrt{2}}{2}\right)=\frac{3\pi}{4}\text{.}\) To avoid this confusion and to ensure the function is one-to-one, we can put restrictions on the domains of each trigonometric function so they attain all the values in the range only once, making the function one-to-one, and thus have an inverse (see FigureÂ 2.4.1).
