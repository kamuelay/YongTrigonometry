Notice that we can rearrange the angular speed to get \(\theta=\omega t\text{.}\) Since \(s\) is an arc length, we have \(s=r\theta\text{,}\) and thus we can write the linear speed as

\begin{equation*} v=\frac{s}{t}=\frac{r\theta}{t}=\frac{r\omega t}{t}=r\omega \end{equation*}
