Solution 3.3.1.1.

To compute \(\cos2\theta\text{,}\) notice there are three different formulas: \(\cos2\theta=\cos^\theta-\sin^2\theta\text{,}\) \(\cos2\theta=1-2\sin^2\theta\text{,}\) or \(\cos2\theta=2\cos^2\theta-1\text{.}\) Using any of the three equations will give us the correct answer, however, if we look at the information we are given, we know that \(\cos\theta=-\frac{4}{5}\) so it may be easier to use \(\cos2\theta=2\cos^2\theta-1\) since the first two equations require us to know \(\sin\theta\text{.}\) Thus

\begin{equation*} \cos2\theta=2\cos^2\theta-1=2\left(-\frac{4}{5}\right)^2-1=2\left(\frac{16}{25}\right)-1=\frac{32}{25}-\frac{25}{25}=\frac{7}{25} \end{equation*}

To compute \(\sin2\theta\text{,}\) \(\tan2\theta\text{,}\) or using the other equations for \(\cos2\theta\text{,}\) we will need to compute \(\sin\theta\) and \(\tan\theta\text{.}\) To do this, we will draw the triangle formed from \(\cos\theta=-\frac{4}{5}\) where \(-\frac{\pi}{2}\lt \theta\lt \pi\text{.}\)

Using the Pythagorean Theorem, we can solve the triangle:

\begin{align*} (-4)^2+y^2\amp =\amp 5^2\\ 16+y^2\amp =\amp 25\\ y^2\amp =\amp 9\\ y\amp =\amp 3 \end{align*}

Using the right triangle, we can now find an expression for

\begin{equation*} \sin\theta=\frac{\mbox{opposite} }{\mbox{hypotenuse} }=\frac{3}{5} \end{equation*}

and

\begin{equation*} \tan\theta=\frac{\mbox{opposite} }{\mbox{adjacent} }=\frac{3}{-4}=-\frac{3}{4} \end{equation*}

Using the Double-Angle Formula, we get

\begin{equation*} \sin2\theta=2\sin\theta\cos\theta=2\left(\frac{3}{5}\right)\left(-\frac{4}{5}\right)=-\frac{24}{25} \end{equation*}

and

\begin{equation*} \tan2\theta=\frac{2\tan\theta}{1-\tan^2\theta}=\frac{2\left(-\frac{3}{4}\right)}{1-\left(-\frac{3}{4}\right)^2}=\frac{-\frac{6}{4}}{1-\frac{9}{16}}=\frac{-\frac{6}{4}}{\frac{16}{16}-\frac{9}{16}}=\frac{-\frac{6}{4}}{\frac{7}{16}}=-\frac{6}{4}\cdot\frac{16}{7}=-\frac{24}{7} \end{equation*}
